# Filmes-TMDB-API

# 🎬 Catálogo de Filmes - TMDB API

Projeto desenvolvido para consumir a API do **The Movie Database (TMDB)** utilizando JavaScript com AJAX (Fetch API), exibindo os filmes mais populares em um layout moderno e interativo.

---

## 🚀 Funcionalidades

- 🔎 Exibição dos **10 filmes mais populares**
- 🎭 Gêneros dos filmes
- ⭐ Avaliação em estrelas
- 📄 Modal com detalhes completos:
  - Poster
  - Título
  - Gêneros
  - Nota
  - Data de lançamento
  - Sinopse
- 🌗 Alternância entre **modo claro e escuro**
- 📱 Layout responsivo
- 📄 Paginação (navegação entre páginas da API)

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API (AJAX)
- API do TMDB

---

## 🔗 API utilizada

Este projeto consome dados da API pública do TMDB:

👉 https://www.themoviedb.org/

> Os dados e imagens são fornecidos pelo TMDB.
